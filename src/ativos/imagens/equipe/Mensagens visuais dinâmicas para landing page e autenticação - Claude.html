Contexto do projeto: plataforma de cursos em TI com React + Vite + CSS puro. Sem biblioteca de UI. A landing page tem múltiplas seções com scroll.

Feature: Toast de mentor por seção (scroll-based)

Implemente um componente <MentorToast /> com as seguintes regras:

Comportamento geral:

  • O toast aparece automaticamente ao entrar na página e ao recarregar
  • Tem um botão de fechar (×)
  • Ao fechar, some e não reaparece na mesma sessão (sessionStorage com chave por seção, ex: mentor_toast_hero_dismissed)
  • Ao trocar de página ou recarregar, o toast aparece novamente (sessionStorage limpa por chave de sessão)
  • Posição: canto inferior esquerdo, fixo, com z-index alto — não use position: fixed dentro de iframe, use portal no React (ReactDOM.createPortal) montado no document.body
  • Animação: fade + slide-up ao entrar, fade + slide-down ao sair

Na Landing Page — muda conforme scroll: Use IntersectionObserver para detectar qual seção está visível. Cada seção tem um id e uma mensagem associada. Quando a seção entra no viewport (threshold 0.4), o toast troca de mensagem com uma animação de transição suave.

Como integrar os SVGs originais do robô

Como você tem os dois arquivos SVG reais (Lado_esquerdo.svg e Lado_direito__1_.svg), a implementação funciona assim:

No componente React:

jsx
import robotLeft from './assets/Lado_esquerdo.svg'
import robotRight from './assets/Lado_direito__1_.svg'

const sectionMessages = [
  { id: 'hero',         pose: 'right', msg: 'Ei! Você está no lugar certo. Essa plataforma foi feita pra pessoas como você.' },
  { id: 'vagas',        pose: 'left',  msg: '80% dos alunos que completam a trilha recebem ao menos uma entrevista em 3 meses.' },
  { id: 'como-funciona',pose: 'right', msg: 'Cada etapa foi pensada pra você não travar. Do zero ao primeiro emprego.' },
  { id: 'quem-somos',   pose: 'left',  msg: 'Conheça o time! Somos pessoas reais que acreditam na sua transição.' },
]

function MentorToast({ message, pose, onClose }) {
  return (
    <div className="mentor-toast">
      <img src={pose === 'left' ? robotLeft : robotRight} width={64} height={64} alt="Mentor" />
      <div className="bubble">
        <span className="label">Mentor</span>
        <p>{message}</p>
      </div>
      <button onClick={onClose}><span>✕</span></button>
    </div>
  )
}

Lógica de scroll com IntersectionObserver:

js
useEffect(() => {
  const obs = new IntersectionObserver(entries => {
    entries.forEach(e => {
      if (e.isIntersecting) {
        const section = sectionMessages.find(s => s.id === e.target.id)
        if (section) setCurrentMessage(section)
      }
    })
  }, { threshold: 0.4 })

  sectionMessages.forEach(s => {
    const el = document.getElementById(s.id)
    if (el) obs.observe(el)
  })
  return () => obs.disconnect()
}, [])

Dica visual: use o robô olhando para a direita (Lado_direito) quando o balão fica à direita dele, e o esquerdo quando ele "vira" para o conteúdo — isso dá a sensação de que ele está apontando para a seção!
